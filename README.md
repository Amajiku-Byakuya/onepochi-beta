# わんぽち
本ソフトウェアのソースコードおよび配布物の無断転載、再配布、改変物の公開、商用利用を禁止します。

## 機能説明
ワンクリックで任意のAPPやブラウザを一括で開きます。

## 詳細機能
- プロファイル機能があり、配信用や仕事用など複数のプロファイルを作成可能です
- PCにインストールされているAPPを一覧で表示し、プロファイルに追加可能
- 任意のURLをプロファイルに追加可能
- PC起動時に自動起動をするかON/OFFできます
- ウィンドウを閉じた後タスクトレイに収納されます
